# Scenario Matrix

Use this matrix to keep test coverage diverse.

## Source shape
- typed topic only
- pasted single-lesson assignment
- pasted weekly assignment list
- pasted outline / TOC
- uploaded image of workbook page
- uploaded image of table of contents
- uploaded PDF excerpt
- mixed source bundle (image + short typed note)

## Household shape
- one learner
- two learners same grade band
- two learners far apart in age
- parent with little time
- parent comfortable improvising
- parent wanting very scripted guidance
- interest-led household
- mixed curriculum household

## Subject family
- math
- reading / language arts
- science
- history / civics
- electives
- logic / chess
- project-based / maker
- intervention / remediation

## Input completeness
- very sparse
- enough for one day
- enough for one week
- enough for a short module
- ambiguous / conflicting
- rich but messy

## Planning horizon expectation
- today only
- today + rough next 2–3 sessions
- week
- module skeleton

## Activity expectation
- no activity needed
- quick practice
- guided parent-led activity
- worksheet-style
- engine-backed interactive activity
- reflection / oral response

## Risk / adversarial categories
- overly broad topic
- inconsistent age signal
- nonstandard pacing
- co-op day constraint
- limited time budget
- parent asks for advanced level beyond learner age
- source uses jargon or poor formatting
- multiple learners sharing one source

## Use this to target gaps

If a scenario fails, label the failure by:
- intake normalization
- horizon inference
- lesson grounding
- activity fit
- parent clarity
- multi-learner context
- mobile usability
- missing pack
